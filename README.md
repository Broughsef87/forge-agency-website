# Forge Agency Website

The B2B AI Automation Agency by Forge OS. 

## Tech Stack
- Next.js (App Router)
- Tailwind CSS v4
- TypeScript
- Framer Motion (planned)

## Brand Aesthetics
- **Core Theme:** Neon-Noir / Cyberpunk Efficiency
- **Primary Color:** Forge Orange (#FF5722)
- **Secondary Color:** Forge Red (#D84315)
- **Background:** Deep Dark (#0A0A0A)

## Offerings
1. **Autonomous Operations:** Custom AI agent workflows for back-office efficiency.
2. **Hyper-Scale Content Ops:** Agentic content production for mid-market firms.
3. **Fractional AI CoS:** 90-day AI transformation strategy.

## Deployment
Deployed to Vercel (Production) via `vercel --prod`.
