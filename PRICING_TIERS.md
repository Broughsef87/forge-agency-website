# Forge Agency Pricing Tiers (2026-03-23)
_Industry standard for ROI-focused bespoke AI agencies_

## Tier 1: Task-Specific Agents
*   **Upfront Build:** $2,000 - $10,000
*   **Monthly Maintenance:** $500 - $1,500
*   **Scope:** Single-purpose agents (e.g., Lead Qualification, Content Ingestion, Support Triage).

## Tier 2: Bespoke Workflow Agents
*   **Upfront Build:** $10,000 - $30,000
*   **Monthly Maintenance:** $2,000 - $5,000
*   **Scope:** End-to-end workflow automation (e.g., Full RevOps Pipeline, Automated Customer Onboarding, Content Production Engine).

## Tier 3: Enterprise Multi-Agent Systems
*   **Upfront Build:** $30,000 - $100,000+
*   **Monthly Maintenance:** $5,000 - $15,000+
*   **Scope:** Interconnected agent swarms, deep internal tool integration, and high-volume data reasoning.

## ROI Model
Forge focuses on **Value-Based Pricing**. If an agent replaces a $60k/year employee, a $15k build + $1k/mo maintenance is a 100% ROI in the first year.
